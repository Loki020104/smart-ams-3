## 🔍 How Eye Blinking Detection Works

### The Algorithm:

```
1. Load image
2. Detect face using dlib frontal face detector
3. Get 68 facial landmarks using shape predictor
4. Extract eye landmarks (36-41 for left, 42-47 for right)
5. Calculate Eye Aspect Ratio (EAR):
   
   Vertical distances:
   - d1 = ||point2 - point6||  (top-bottom)
   - d2 = ||point3 - point5||  (top-bottom)
   
   Horizontal distance:
   - d_h = ||point1 - point4|| (left-right)
   
   EAR = (d1 + d2) / (2 × d_h)

6. Check if EAR ≥ 0.1:
   ✓ YES → Eyes OPEN (live person)
   ✗ NO  → Eyes CLOSED (fake image)
```

### What It Prevents:

✓ Static photographs with closed eyes  
✓ Winking (single eye closed)  
✓ Printed/displayed images  
✓ Partially obscured faces (eyes covered)  
✓ Face turned away (eyes not visible)
